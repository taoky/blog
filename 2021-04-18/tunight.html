THU 半日游：参加 tunight 杂记

Apr 18, 2021 • life • tuna, tunight, thu

博客已经接近两年没更新了。从现在的眼光来看，我写的技术文章不多，质量也不算高，但是在写博客方面，遇到的困境就是「忙的时候没空写，闲的时候懒得（敲键盘）写」。我快本科毕业了，从写上一篇博文开始（2019/07/08），世界发生了太多的变化，自己也获得了一大堆坑需要填上。争取之后更新的频率能提高一些。

我的 Slides

THU 比肥科大多了，整体的体验也挺好的，详细的杂记部分争取暑假的时候能够补上。

2021/08/01

来填坑了。以下根据手头的聊天记录等回忆得到，可能和实际情况有一些出入。

大四这十个多月（2020/09 - 2021/07）我都在北京的 MSRA 实习。本来想，既然到了北京，那肯定要去 TUNA 总部面一面，听一次 tunight 吧。但是诸位都知道 2020 年（初）发生了啥，没法以游客的身份进学校，于是这个想法也就只能搁置下来。今年年初的时候，TUNA 的 @ZenithalHourlyRate (下面简称为 Z 同学) 开始做 mirrorz 项目，作为肥科镜像站的维护者之一我也差不多在那个时间段认识了 Z 同学。三月底的时候我，@iBug, Z 和 TUNA 的杰哥去五道口的日昌饭店（根据不完全归纳，发现 THU 的同学很喜欢这家饭店）约了个饭，之后几周就决定去讲 tunight 了。

4/17 下午，由 iBug 带路，我们两个来到了 THU 的东南门（一开始还走错方向了），与 Z 同学顺利会合。由于我不会骑车，于是我们的交通方式被迫降级，下午把 THU 一部分地方走了一遍。（挺大的，感觉不会骑车的话平时上课什么的确实会有点难受）

恰逢 THU 的 110 周年校庆，进来的时候我被一个老大爷拉住，希望我来帮他们拍照。

这个……我忘了叫啥，好像是东南门那里的礼堂？

Harry: 是中央主楼，一楼和二楼有大报告厅，其他是办公区域。

「清华园」门 ，如果我没记错的话应该是个模型。

Harry: 是二校门，不是模型，是文革后重建的。

两张风景照。第一张是「荷塘月色」，据称是游客必去景点。

当时我感觉好像我在科大也见到过长得和这差不多的，不过后来翻照片发现两者的相同之处只有心形。对了，THU CS 可以重排成 USTC H

有历史感的老图书馆。

晚饭时间。一个游客不需要刷校园卡的餐厅，香锅还挺好吃的。

吃完晚饭之后到法图。虽然来的人不算很多，但是氛围很好。TUNA 的康哥（希望我没有认错人）向我们展示了「笔记本斯坦尼康」（这个名字其实是我们开玩笑起的）。

卷王必备，即使不在自行车上，也可以操作你的笔记本。（图片有裁剪）

然后 tunight 环节，我先讲（slides 在本文开头链接），然后 iBug 讲。这次因为比较即兴，所以 slides 的 framework 用的也是比较即兴的 remark（不是处理 Markdown 的 remark.js，不要搞混了！）。直接一个 HTML 开写 markdown 确实挺舒服的……

我讲的部分主要整理了两个 debug 的（还算有点意思的）经历：一个和内核处理网络的 bug 有关，另一个和 GRUB 处理 LVM metadata 的 bug（这个 bug 上游还没人来修）有关。其实这个 topic 当时也想了很长时间，因为我也不算擅长讲 talk，最后想了很久才觉得这个 topic 可能可以讲点啥出来（当时也刚好处理好了 GRUB 的 bug）。iBug 的 topic 比较杂，大概介绍了 LUG 的内网架构等等。

用 Zoom 来讲，体验确实不错，可惜现在 Zoom 的 license 国内没那么容易获得了。我们两个讲完之后，差不多就进入了闲聊模式。各位同学随便聊了一些话题。现在回忆的话（隔了三个多月忘了很多），大概扯到了：

  • CSS 以「水」字为一个长度单位的实验性标准（现在没有浏览器实现）
  • Base1k
  • 妮可草分享自己的博客（见下图），以及和 wireguard 内网相关的一些东西。

最后合了个影，愉快结束了。（虽然脸盲的我，还是没有记住谁是谁。）
